小朋友使用手冊
澄萌你理財寶貝 · 歡迎使用！🐻
🏠 首頁
📌 🐻 澄萌財富樹：你的專屬小樹！存越多錢、賺越多分，樹就會長大。點樹可以看「財富日記」。
📌 今日收益：今天賺了多少分，點它可以看詳細明細。
📌 種樹天數：你種樹種了幾天了。
📌 🐻 我的帳戶：可用帳戶、銀行、債券、股票，點每個帳戶可以看詳細記錄。
📌 我的夢想倒數：你設定的夢想目標進度，點進去可以存錢。
📌 今日任務：今天要完成的任務，最多顯示5個。
🐻 任務打卡
📌 完成任務後，按中間的 🐻 任務打卡 按鈕。
📌 選擇你完成的任務，送出申請。
📌 申請後會顯示「⏳ 待確認」，要等爸媽在後台確認才會加分。
📌 每個任務每天只能申請一次喔！
📌 連續打卡可以獲得特別獎勵：
   🌱 連續3天 +5分
   🔥 連續7天 +15分
   ⭐ 連續14天 +25分
   👑 連續30天 +50分
💰 存錢投資
📌 凱莉銀行：最安全的存法，年利率5%，每天都有一點點利息！
📌 媽媽債券：6種債券可以選，利率比銀行高，但有到期日。
📌 爸爸美股：6支股票，有些有配息，有些只有成長。
📌 高利率 = 高風險！ 不是利率越高越好，記住這個道理。
📌 分散投資最安全，不要把所有錢放在同一個地方。
📌 存入後，分數會從「可用帳戶」轉移到投資帳戶。
🎁 兌換
📌 兌換現金：10分 = RM1，讓爸媽換給你真實的錢。
📌 平板使用時間：10分 = 10分鐘。
📌 電動遊戲：告訴系統你想玩幾分鐘，自動計算需要幾分，並開始倒數計時！
📌 購買玩具：申請後讓爸媽決定要扣多少分。
📌 兌換後 10分鐘內 可以反悔取消，分數會退回。
🎯 夢想存錢筒
📌 設定你的夢想目標（例如：買腳踏車需要500分）。
📌 可以設定目標日期，系統會告訴你每天需要存幾分。
📌 夢想筒沒有利息，但存進去的分數是你的！
📌 達到目標後會有慶祝動畫和夢想徽章 🏆
📌 最多可以同時有多個夢想目標。
📚 理財教室
📌 共 17 堂課，教你各種理財知識。
📌 每堂課後有小考，答對可以獲得獎勵分！
📌 熊熊信箱：有問題可以傳訊息給爸媽，熊熊理財專家會回覆你。
📌 課程涵蓋：被動收入、股票、債券、配息、風險、分散投資等。
📊 財富報告
📌 圓餅圖：看看你的錢分配在哪裡。
📌 今日收益明細：今天各來源賺了多少。
📌 7天走勢圖：你的總分變化趨勢。
📌 投資成績：每個帳戶的投入 vs 現值。
📌 本月任務：這個月打了幾次卡、賺了多少分。
📌 配息倒數：距離下次配息還有幾天。

爸媽使用手冊
後台完整操作說明
🔐 進入後台
📌 點畫面右上角的 🔐 按鈕。
📌 輸入後台密碼（預設：1234）。
📌 建議在「設定」分頁修改為自己的密碼。
📌 後台共有 7 個分頁：分數、任務、利率、信箱、設定、版本、事件。
👤 分數分頁
📌 待確認任務：孩子打卡後，在這裡按「✓ 確認」才會加分，或「✗ 退回」請孩子重做。
📌 手動加減分：輸入分數按 ➕ 或 ➖，可直接調整分數。
📌 修改帳戶餘額：直接編輯各帳戶數字，按「儲存」。
📌 改名：輸入新名字按「確認」。
📌 重置資料：清除孩子所有資料，請先備份！
📋 任務分頁
📌 頂部切換不同孩子，每個孩子有獨立任務清單。
📌 新增任務：輸入名稱和分數，按「新增任務」。
📌 編輯任務：按 ✏️ 修改名稱和分數。
📌 刪除任務：按 🗑️ 刪除。
📌 建議設定多種任務，讓孩子有多元選擇。
💹 利率分頁
📌 設定銀行年利率（建議 3-8%）。
📌 設定債券年利率（建議 4-10%）。
📌 可以參考現實中的利率新聞來調整，讓孩子學習現實連結。
📌 調整後按「儲存利率」即生效，新的利息從隔天開始計算。
💬 信箱分頁
📌 孩子在「理財教室 → 熊熊信箱」提問後，這裡會顯示訊息。
📌 頂部選擇要回覆哪個孩子，輸入回覆內容，按「回覆」。
📌 孩孩子下次打開理財教室就能看到熊熊的回覆。
📌 可以用來解答理財問題，也可以傳鼓勵的話 💪
⚙️ 設定分頁
📌 修改密碼：輸入新密碼按「修改」，下次登入使用新密碼。
📌 新增孩子：最多 5 位，輸入名字和生日，選擇顏色主題。
📌 孩子管理：
   🎂 設定/修改生日
   🎨 更換顏色主題（6種）
   🔒 設定/修改 PIN 碼（孩子切換帳號時需要輸入）
   ✕PIN 清除 PIN 碼
   🗑️ 刪除孩子帳號
🔄 版本分頁（備份）
📌 手動備份：按「匯出備份檔案」下載 JSON 檔案到手機。
📌 Google Drive 備份：連線後按「立即備份到 Drive」，備份到雲端。
📌 恢復備份：選擇備份檔案，或從 Drive 恢復最新備份。
📌 ⚠️ 建議每週備份一次，以防資料遺失！
📌 更新 App（重新上傳）前一定要先備份。
⚡ 事件分頁（風險教育）
📌 這個功能讓你「模擬」真實市場事件，讓孩子學習風險。
📌 觸發流程：選擇投資→選擇事件類型→輸入幅度→填新聞標題→按觸發。
